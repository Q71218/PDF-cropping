PDF 多選區裁切工具 (10x15cm) 使用說明
設計：DY

─────────────────────────────
📁 主要功能簡介：
─────────────────────────────

本工具可協助您從 PDF 文件中選取多個裁切區域，並匯出成單獨的新 PDF 頁面。特別適用於條碼、標籤、票券等固定區塊裁切需求。

─────────────────────────────
📌 操作流程：
─────────────────────────────

1. 點擊「開啟 PDF」按鈕，選取要處理的 PDF 文件。
2. 於畫面中以滑鼠拖曳方式選取裁切區域（可選多個）。
3. 如需取消所有選取，點擊「清除選取區」。
4. 可點擊「返回上一步」來移除最後一次的選取。
5. 點擊「上一頁」或「下一頁」切換頁面（會清除當頁已選區域）。
6. 選擇是否要勾選「批次多頁處理」：
   - ✔ 勾選：每頁皆裁切相同位置。
   - ☐ 不勾選：僅裁切目前頁面。
7. 點擊「匯出裁切區」將選取區域輸出為新 PDF（10x15cm）。
   - 圖片將自動置中、轉為 300dpi。
   - 每頁底部會加註「拆封請全程錄影」的紅色提示字。

─────────────────────────────
🖱 滑鼠操作：
─────────────────────────────
- 滑鼠左鍵按住拖曳：建立裁切區
- 拖曳順序將決定匯出時的頁面順序
- 每個選取區中間會顯示編號

─────────────────────────────
📐 裁切尺寸說明：
─────────────────────────────
- 輸出 PDF 每頁尺寸為 10 x 15 公分（相當於 3R 照片大小）
- 圖片將自動縮放置中，不會裁切變形

─────────────────────────────
💡 額外說明：
─────────────────────────────
- 若遇密碼保護 PDF，將提示輸入密碼
- 若裁切區域為空白（白色 >98%），則該區將不匯出
- 匯出的檔名為「原檔名_已裁切.pdf」，儲存在原檔相同資料夾

─────────────────────────────
🖥 執行環境需求：
─────────────────────────────
- Python 3.x
- 需安裝以下套件：
    pip install pymupdf pillow

─────────────────────────────
📦 封裝為 EXE 指令（可選）：
─────────────────────────────
pyinstaller --noconsole --onefile --add-data "fonts;fonts" your_script.py

─────────────────────────────
📌 注意事項：
─────────────────────────────
- 若要封裝中文字型，請將字型放入 `fonts` 資料夾並載入
- 請確保 PyMuPDF 採用版本支援圖片裁切與插入
