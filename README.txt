PDF 多選區裁切工具（10x15cm）  設計：DY
----------------------------

📌 功能簡介
本工具可讓使用者載入 PDF 檔案，透過滑鼠手動選取多個裁切區域，並匯出為 10x15cm 的 PDF 頁面（適合相片列印大小）。可支援多頁處理與自動過濾空白區域。

📥 開啟 PDF
點選「開啟 PDF」按鈕

選擇一個 PDF 檔案

若檔案有密碼，系統會提示輸入

✂️ 選取裁切區域
使用滑鼠左鍵在畫面上「按住拖曳」選取區域

放開滑鼠左鍵即可儲存選取區域

每個區域會以藍框標示，中央有紅色編號（順序）

可選取多個區域（依序匯出）

🔁 操作按鈕說明
按鈕名稱	說明
開啟 PDF	開啟一個 PDF 檔案
匯出裁切區	依選取區匯出裁切 PDF 頁面
上一頁	檢視上一頁 PDF
下一頁	檢視下一頁 PDF
清除選取區	移除所有已標記的裁切區域
批次多頁處理	啟用後將套用相同裁切區到每一頁

📤 匯出裁切區域
點選「匯出裁切區」後會建立新的 PDF 檔案，儲存在原始 PDF 相同的資料夾中

每個裁切結果都會輸出為 10x15cm 大小（300 DPI）

每頁下方會自動加入紅色文字：「拆封請全程錄影」

空白頁（畫面幾乎全白）將自動跳過不輸出

🧠 補充說明
開啟檔案後預設顯示大小為原始的 90%

支援中文 PDF、加密 PDF（需輸入密碼）

使用者可在程式視窗中自由調整大小，圖像會自動縮放顯示
